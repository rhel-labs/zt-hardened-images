= UBI Baseline

Red Hat Hardened Images (HI) are a catalog of container images designed for production use: smaller attack surface, non-root by default, rebuilt rapidly when CVEs are patched.
This lab runs the same Flask application on a UBI base, a hardened base, a FIPS-enforcing base, and behind a TLS reverse proxy.
This module establishes the UBI baseline.

== The Application

The lab app is `rhhi-demo`, a small Flask web app pre-staged in `~/flask/`.
It serves a crypto demo at `/crypto-demo` that hashes submitted text using MD5, SHA-256, and SHA-512.

[source,bash,role=execute]
----
ls ~/flask/
----

You'll see the application code (`app.py`) and three Containerfiles: `Containerfile.ubi`, `Containerfile.hardened`, and `Containerfile.fips`.

== The UBI Containerfile

[source,bash,role=execute]
----
cat ~/flask/Containerfile.ubi
----

`ubi9/python-312` is Red Hat's general-purpose Python 3.12 base.
Pip is pre-installed; Flask installs from external PyPI with an explicit `USER root`.
A non-root runtime user requires an explicit `USER` directive — UBI does not enforce non-root by default.

This Containerfile is the conventional approach.
Keep it in mind as the comparison point for the rest of the lab.

== Run the Application

`rhhi-demo:ubi` has been pre-built.

[source,bash,role=execute]
----
podman run -d --name rhhi-ubi -p 8080:8080 rhhi-demo:ubi
----

[source,bash,role=execute]
----
echo "http://flask-{guid}.{domain}/crypto-demo"
----

Navigate to that URL, enter some text, and click *Hash*.
All three algorithms — MD5, SHA-256, SHA-512 — return a hex digest.
MD5 succeeds here.
Keep that in mind.

== What's in the Image?

[source,bash,role=execute]
----
podman run --rm rhhi-demo:ubi rpm -qa | wc -l
----

UBI Python includes packages that cover a broad range of use cases: build tools, development headers, documentation.
That breadth is convenient for development but is surface area a production runtime does not need.

== Stop the Container

[source,bash,role=execute]
----
podman stop rhhi-ubi && podman rm rhhi-ubi
----
